# clawcheckin
使用 Python + Playwright 实现自动登录、自动签到、Telegram 推送、自动重试、随机延迟防封，并可通过 GitHub Actions 自动每天签到。
✨ 功能特性

✅ 基于 Playwright（比 Selenium 更快、更稳定）

✅ 自动登录 Claw Cloud

✅ 自动签到

✅ 自动判断是否签到成功

✅ 自动重试（默认 3 次）

✅ 随机延迟防封（完全模拟人工操作）

✅ Telegram 推送签到结果

✅ GitHub Actions 定时运行，无需服务器

✅ 完整项目可直接 Push 到 GitHub

| Secret Key     | Value              |
| -------------- | ------------------ |
| `USERNAME`     | Claw 登录邮箱      |
| `PASSWORD`     | Claw 登录密码      |
| `TG_BOT_TOKEN` | Telegram Bot Token |
| `TG_CHAT_ID`   | Telegram Chat ID   |
